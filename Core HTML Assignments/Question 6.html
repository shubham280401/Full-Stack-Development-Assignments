1. HTML

    2. Primary Use Case Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browser
    • Allows for semantic markup
    Form Input Handling
    • Allows the creation of interactive forms with input fields, buttons,and validation.
    • Utilizes form elements like <form>,<input>,<button>
    • Supports various input types such as text, radio, checkbox,etc.

  2. CSS

    3. Primary Use Case Key Features/Benefits
    Styling Web Pages
    • Styling and layouts of HTML elements.
    • Seperation of content and presentation
    • Control over fonts, colour, spacing, etc.
    Responsive Web Design
    • Media queries for different screen sizes
    • Flexible and grid layouts
    • Viewport unit for responsive design.

  3. JavaScript

    4. Primary Use Case Key Features/Benefits
    Adding Interactivity
    • Enables interactive elemnts in web pages.
    • Response to user inputs like clicks, events and more.
    • Manipulates HTML and CSS dynamically.
    Data Manipulation
    • Stores and manages data using variables.
    • Performs calculations and logic operations.
    • Access and modifies web page contents.
