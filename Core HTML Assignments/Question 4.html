Tags in HTML

  1. <!DOCTYPE html>: It is used specify which version of HTML is being used
  2. <h1> to <h6>: It is used to give headings.
  3. <img>: It is used to define an image.
  4. <em>: It is used to make text italic.
  5. <strong>: It is used to make the text bold.
  6. <br>: It is used for line break.
  7. <p>: It is used to define paragraph.
  8. <u>: It is used to make underline.
  9. <th>:It is used to define header cell in table.
  10. <tr>: It is used to define row in a table.
  11. <title>: I t is used to define title of doocument.
  12. <td>: It is used to define cell in a table.
  13. <ul>: It is used to define unordered list.
  14. <ol>: It is used to define ordered list.
  15. <li>: It is used for list item.
